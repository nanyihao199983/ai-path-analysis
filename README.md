# 📊 AI 路子大数据分析系统 | 1324 实战版

> AI 驱动的路子数据分析平台 — 1324 投注策略 + 智能预测 + 资金规划

## ✨ 功能特性

- **💰 资金规划** — 智能资金管理，投入与收益分析
- **📊 路子采集** — 实时记录历史路子数据
- **🤖 AI 分析** — 大数据建模，走势智能预测
- **🎯 1324 策略** — 科学投注方案，风险控制
- **📱 移动端** — 完美适配手机 H5
- **📈 可视化** — 专业级数据可视化图表

## 🚀 在线体验

- **在线访问:** https://nanyihao199983.github.io/ai-path-analysis/

## 🛠️ 技术栈

- **前端:** HTML5 / CSS3 / JavaScript (ES6+)
- **部署:** GitHub Pages / Netlify
- **架构:** 纯前端 SPA，零后端依赖

## 📦 本地运行

```bash
git clone https://github.com/nanyihao199983/ai-path-analysis.git
cd ai-path-analysis
# 直接用浏览器打开 index.html 即可
```
